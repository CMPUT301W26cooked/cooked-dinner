# UI Decisions

This folder stores UI/UX decisions and open questions so the team stays consistent.

## What goes here
- Decision records describing what we chose and why
- Open questions that need team/TA input

## Naming convention (recommended)
- `decision_<topic>.md`
  - Example: `decision_navigation_structure.md`

## Decision template (copy/paste)
```md
# Decision: <short title>

## Date
MM-DD

## Context
(What problem are we solving?)

## Options Considered
- Option A:
- Option B:
- Option C:

## Decision
(What we chose)

## Rationale
(Why this option)

## Consequences / Follow-ups
(What changes because of this?)
