
# UI Mockups

Put UI screen mockups in this folder.

## What goes here
- PNG (preferred) or PDF exports of key app screens
- “Major dialogs” / important screens used in storyboards

## Naming convention (recommended)
- `screen_<feature>_v<version>.png`
  - Example: `screen_event_details_v1.png`

## How to export from Figma (quick)
1. Select the frame
2. Right sidebar → **Export**
3. Export as **PNG** (2x recommended)
4. Save into this folder
